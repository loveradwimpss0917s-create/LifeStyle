# 01. 要件定義・MVP定義

## 1. 要件の全体像

要件は3系統に分かれる。実装順もこの順とする。

| 系統 | 内容 | 実装時期 |
|---|---|---|
| A. サイト(公開面) | ブランドサイト本体。訪問者が見るすべて | V1(MVP) |
| B. コンテンツ基盤(編集面) | 商品・記事・SNS派生コンテンツのデータ構造と入稿フロー | V1(MVP)〜V2 |
| C. AIパイプライン(自動化面) | 入力→解析→生成→承認→公開→再利用の自動化 | V2〜V4(段階導入) |

## 3. 非機能要件

| N-02 | アクセシビリティ | WCAG 2.2 AA。コントラスト比4.5:1以上、全画像にalt、キーボード操作可、`prefers-reduced-motion`対応 |
| N-03 | SEO | 全ページ静的HTML。構造化データ(Product/Article/BreadcrumbList/FAQ)。サイトマップ・RSS自動生成 |
| N-04 | レスポンシブ | 375px〜1920px。モバイルファースト(流入の8割はSNS経由=スマホ想定) |
| N-05 | 保守性 | コンテンツ追加にコード変更不要。スキーマはZodで型検証しビルド時にエラー検出 |
| N-06 | 可搬性 | コンテンツはMarkdown/JSON。フレームワーク移行時もコンテンツ資産は無傷 |
| N-07 | コスト | V1の月額ランニングコスト0円(Cloudflare Pages無料枠/GitHub無料枠)。V2以降もAI API費のみ(目安 月$5〜20) |

## 4. MVP定義(V1)

### 4.1 MVPの目的

**「世界観が完成したブランドサイトを最短で公開し、既存Instagram流入の受け皿を作る」**。
AI自動化はMVPに含めない(コンテンツ初期投入はAI支援の手動で行う)。理由: 自動化パイプラインは「流し込む先」が完成していないと設計が空転する。先に器を完成させる。

### 4.2 MVPスコープ

**含む**:
- サイト全8画面(トップ/カテゴリ/商品/記事/プロフィール/検索/ランキング/タグ)
- デザインシステム一式(トークン・コンポーネント)
- コンテンツスキーマ(商品・記事)と初期コンテンツ(商品10件・記事6本を目標)
- SEO基盤(メタ・構造化データ・サイトマップ・RSS・OGP自動生成)
- アフィリエイトリンクコンポーネント(PR表記込み)
- Cloudflare Pages自動デプロイ(GitHub push → build → deploy)

**含まない**(明示的にスコープ外):
- AI自動解析・自動生成(V2)
- お気に入り・比較ページ・ニュースレター(V2)
- 自前アクセス解析(V3)
- Instagram API連携(V3)
- 会員機能・コメント機能(永久にスコープ外)

### 4.3 MVP完了の受け入れ基準

1. 商品Markdownファイルを1つ追加してpushすると、商品ページ・カテゴリ一覧・検索・サイトマップに自動反映される
2. Lighthouseで Performance/Accessibility/Best Practices/SEO すべて95以上
3. スマホ実機でトップ→カテゴリ→商品→Yahoo!ボタンまで3タップ以内
4. すべてのアフィリエイトリンク含有ページにPR表記が表示される
5. OGP画像がSNSシェア時に正しく表示される

### 4.4 工数見積(AI実装エージェント前提)

| フェーズ | 内容 | 目安 |
|---|---|---|
| 1 | プロジェクト雛形・デザイントークン・基本レイアウト | 1セッション |
| 2 | コンテンツスキーマ・サンプルデータ・商品/記事ページ | 1〜2セッション |
| 3 | トップ・カテゴリ・ランキング・タグ・プロフィール | 1〜2セッション |
| 4 | 検索(Pagefind)・SEO一式・PWA manifest | 1セッション |
| 5 | 仕上げ(a11y監査・Lighthouse調整・デプロイ設定) | 1セッション |

人間が並走する作業: サイト名・ドメイン決定、初期商品データのメモ書き、写真の提供、Yahoo!アフィリエイトID取得。

## 5. 制約条件

| 制約 | 内容 |
|---|---|
| 運用者は1人 | 承認・入稿以外の定常作業を発生させてはならない |
| 入力はストーリー粒度 | 「写真数枚+短いメモ」から全コンテンツが生成できる設計であること |
| プラットフォーム規約 | Instagram自動投稿はGraph API公式手段のみ検討(スクレイピング・非公式API禁止)。TikTok/YouTubeは台本・素材生成までとし投稿は手動(V4で公式API検討) |
| Yahoo!アフィリエイト規約 | リンク改変禁止・PR表記必須・成果計測パラメータは規約準拠のこと |
| 予算 | 初期費用ほぼゼロ。ドメイン代(年約1,500円)+ V2以降のAI API費のみ |
